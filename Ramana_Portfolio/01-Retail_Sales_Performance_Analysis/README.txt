# Retail Sales Performance Analysis

## Project Overview

This project analyzes a retail sales dataset to uncover business insights related to sales performance, profitability, regional sales distribution, and product performance.

The project demonstrates an end-to-end **Data Analyst workflow** using SQL for data analysis, Excel for exploratory analysis, and Power BI for interactive dashboards.

---

## Tools Used

* SQL (MySQL Workbench)
* Microsoft Excel
* Power BI

---

## Dataset Information

Dataset: Sample Superstore
Records: ~9K+ rows

Key Columns:

* Order Date
* Category
* Region
* Segment
* Product Name
* Sales
* Profit
* Discount

---

## Project Workflow

### 1. SQL Data Analysis

* Imported dataset into MySQL Workbench
* Performed SQL queries to analyze sales performance
* Calculated total sales, total profit, and order count
* Analyzed sales by category, region, and customer segment
* Identified top-performing and loss-making products

---

### 2. Excel Data Analysis

* Created pivot tables for sales insights
* Built charts to analyze category performance
* Analyzed regional sales distribution
* Developed a dashboard using pivot charts and slicers

---

### 3. Power BI Dashboard

* Built an interactive sales dashboard
* Created KPI cards for Total Sales, Total Profit, and Total Orders
* Visualized sales by category and region
* Added slicers for dynamic filtering
* Analyzed monthly sales trends

---

## Key Insights

* Technology category generated the highest revenue.
* West region contributed the highest share of sales.
* Consumer segment generated the majority of orders.
* High discounts led to losses for certain products.
